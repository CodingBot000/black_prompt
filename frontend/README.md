# Black Prompt MVP

Black Prompt는 허구의 디지털 인프라 침투/AI 방어 전략 시뮬레이션입니다. 실제 사이버 공격, 네트워크 스캔, 익스플로잇, 데이터 수집 기능은 포함하지 않으며 모든 진행은 추상화된 게임 수치로만 처리됩니다.

## 실행

```bash
npm install
npm run dev
npm test
npm run build
```

개발 서버 기본 주소는 `http://127.0.0.1:5173/` 또는 Vite가 출력하는 로컬 주소입니다.

## 조작

- 새 게임에서 시작 권역을 선택합니다.
- 지도 위 권역 마커를 눌러 상태를 확인합니다.
- 하단 작전 버튼으로 선택 권역에 즉시 작전을 실행합니다.
- 스킬트리, 글로벌 상태, 이벤트 로그, 도움말은 하단 버튼으로 엽니다.
- 재생/정지 버튼으로 날짜 진행을 제어하고, 속도 버튼으로 1x/2x를 전환합니다.
- 스킬트리/상태/이벤트/도움말/새 게임/엔딩 모달이 열리면 시간이 자동으로 멈춥니다. 모달을 닫아도 자동 재생하지 않습니다.

## 구조

- `src/data`: 8개 권역, 6개 작전, 15개 스킬, 10개 이벤트, 방어 프로토콜, 엔딩 메타데이터
- `src/engine`: 순수 함수 기반 게임 상태 생성, 수식, tick, 작전/스킬/이벤트/방어/승패 처리
- `src/locales`: 한국어 locale과 영어 fallback 구조
- `src/components`: 지도, 상태바, 패널, 모달 UI
- `src/state`: `useReducer` 연결, 자동 저장, localStorage 저장/불러오기, 실시간 루프
- `src/tests`: Vitest 기반 엔진/데이터 테스트

## 검증

현재 구현은 `npm test`와 `npm run build`를 통과합니다. Playwright 기반 브라우저 검증 산출물은 `output/web-game/`에 저장됩니다.
